# Plific

MVP de demonstração de um produto de antecipação de recebíveis para
trabalhadores da gig economy (motoristas de app, entregadores e autônomos).

> ⚠️ Este é um ambiente de teste. Nenhum dado real é coletado e nenhuma
> transação financeira de verdade acontece.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS 4

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy automático na Vercel a cada push na branch `main`.
